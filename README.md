## Getting started

Installation
```bash
yarn add ghostmarket-sdk-ts
```
or if using npm
```bash
npm install --save ghostmarket-sdk-ts
```

## Development

**Setup**

Clone this repo
```bash
git clone https://github.com/OnBlockIO/ghostmarket-sdk-ts
```

Install 
```bash
cd ghostmarket-sdk-ts/
yarn install
```

**Build**

To build
```bash
yarn build
```

**Test**

To run the tests
```bash
yarn test
```
